<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Feature data to data-display slots | Tenant-owned data enters render slots; UI kit must not fetch, filter, or transform tenant scope. |
| Status color to user perception | Status cannot rely on color alone. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-06 | Information Disclosure | `UiTable`, `UiList`, `UiMetricCard` | mitigate | Components render caller content only and contain no API/composable/store calls. |
| T-02-07 | Tampering | `UiBadge`, `UiStatusDot` | mitigate | Use escaped Vue text/slots; avoid `v-html`; pair colors with text. |
| T-02-08 | Denial of Service | `UiTable` responsive overflow | mitigate | Horizontal overflow wrapper and stable row heights prevent clipped table controls. |
| T-02-SC | Tampering | npm package surface | mitigate | No package install in this plan. |
</threat_model>
