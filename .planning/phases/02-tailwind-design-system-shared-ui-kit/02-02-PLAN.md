## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Caller form state to Ui controls | Values/errors may originate from user input and server validation; controls only display external state. |
| Keyboard user to controls | Interactions must remain reachable without mouse-specific behavior. |
